Assignment targets must be an Entra ID group. --- ## 8. What the Scripts Do ### Install-BatchFileRenamer.ps1 1. Creates `C:\Program Files\WilmerHale\Batch File Renamer\` if it does not exist 2. Copies `BatchFileRenamer.exe` from the package source to the install directory 3. Creates an All-Users Start Menu shortcut at `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\BatchFileRenamer.lnk` 4. Exits 0 on success, 1 on any failure ### Detect-BatchFileRenamer.ps1 - Checks for `C:\Program Files\WilmerHale\Batch File Renamer\BatchFileRenamer.exe` - Exits 0 (detected) or 1 (not detected) ### Uninstall-BatchFileRenamer.ps1 1. Removes `BatchFileRenamer.exe` from the install directory 2. Removes the Start Menu shortcut 3. Removes `C:\Program Files\WilmerHale\Batch File Renamer\` if empty; leaves parent `C:\Program Files\WilmerHale\` intact 4. Exits 0 on success, 1 on any failure --- ## 9.
